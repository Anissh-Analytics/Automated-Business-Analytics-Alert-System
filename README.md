# 📊 Automated Business Analytics & Alert System

An end-to-end automated analytics pipeline that processes business data, generates insights, and triggers alerts — simulating a real-world data analytics system.

---

## 🚀 Overview

This project demonstrates how raw business data can be transformed into actionable insights using automation, analytics, and decision logic.

The system:
- Fetches data from Excel 📂  
- Processes key business metrics 📊  
- Detects performance issues 🚨  
- Sends alerts via email 📧  
- Generates business insights 🤖  

---

## 🧠 Key Features

### 🔹 Automated Data Pipeline
- Built using **n8n**
- Scheduled execution to simulate daily business monitoring
- Reads and processes Excel-based datasets automatically

---

### 🔹 KPI Computation Engine
- Calculates key metrics:
  - 💰 Revenue  
  - 📈 Profit  
  - 📊 Average Profit  
  - 📦 Total Orders  

---

### 🔹 Alert System 🚨
- Rule-based logic to detect anomalies
- Triggers **email alerts** when profit drops below threshold
- Helps in proactive decision-making

---

### 🔹 Insight Generation Engine 🤖
- Generates **dynamic business insights**
- Based on:
  - Profit margin
  - Revenue thresholds
  - Order efficiency
- Simulates AI-driven analytics using rule-based logic

---

### 🔹 Power BI Integration 📊
- Dashboard built for:
  - Visualizing KPIs
  - Tracking performance trends
- Enhances decision-making with visual insights

---

### 🔹 Data Cleaning & Transformation 🧹
- Handles inconsistent Excel data
- Implements preprocessing logic inside workflow
- Ensures reliable and accurate analysis

---

## ⚙️ Tech Stack

- **n8n** – Workflow Automation  
- **Excel** – Data Source  
- **Power BI** – Data Visualization  
- **JavaScript** – Data Processing Logic
